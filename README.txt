In order to run this application, you will need to run:
npm install
in the following folders:
- server
- ui
- main project folder

You will also need to set up a local sql database with the file 'UserDatabaseInitial'
You will also need to edit the .env file within /server directory to match the local SQL server

after you have installed and edited these files, run 
node server.js
in the server directory and
npm start
in the ui directory

this will open a browser that is running a localhost react website.
